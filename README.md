# roll

> A dice roller / random number generator


## Install

``` bash
$ yarn global add @nothingrandom/roll
$ npm install --global @nothingrandom/roll
```


## Usage

``` bash
$ roll --help

  Usage
    $ roll <dice to roll>

  Example
    $ roll d20
    10
```
